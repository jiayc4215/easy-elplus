## easy-elplus 是什么？

easy-elplus 是一个基于 [Element Plus](https://element-plus.org/) 的增强组件库，旨在简化开发流程，提升开发效率。

## 特性

- 🚀 **开箱即用** - 基于 Element Plus 封装，提供更便捷的 API
- 📦 **按需引入** - 支持 Tree-shaking，减小打包体积
- 🎨 **高度可定制** - 灵活的配置和插槽系统

## 组件列表

目前 easy-elplus 提供以下组件：

### 基础组件

- **Button** - 增强的按钮组件，支持加载状态等
- **Dialog** - 对话框组件，简化使用方式

### 数据展示

- **Table** - 表格组件，支持多级表头、选择、操作列等
- **Descriptions** - 描述列表组件，支持分组和动态渲染

### 表单组件

- **ImageUpload** - 图片上传组件
- **FileUpload** - 文件上传组件

## 为什么选择 easy-elplus？

Element Plus 是一个优秀的 Vue 3 组件库，但在实际业务开发中，我们经常需要对组件进行二次封装以满足特定需求。easy-elplus 正是基于这一需求诞生的，它提供了：

1. **更简洁的 API** - 减少重复配置，提高开发效率
2. **业务场景优化** - 针对常见业务场景进行优化
3. **统一的使用体验** - 保持一致的 API 设计风格
